pimple php的ioc容器

pimple是一款很简单的php依赖注入容器,它实现了php的ArrayAccess的接口.很好地提高了程序的灵活性以及实现了对象的延迟加载.

ArrayAccess是一个提供像访问数组一样访问对象的能力的接口.

官方手册

首先先建立一个container容器

use Pimple\Container;

$container = new Container();

然后在容器中添加匿名函数

// define some services
$container['session_storage'] = function ($c) {
    return new SessionStorage('SESSION_ID');
};//自动调用offsetSet

调用session对象

$session = $container['session'];

PHP匿名函数

php从5.3即开始支持闭包函数,在项目中还是经常用到的

##举个例子

$add = function($num1, $num2){
    return $num1 + num2;
};
array_map($add, arr1, $arr2);

如果要用到上下文的变量,则需要使用use关键字

$name = 'sunny';
$func = function() use ($name){
    echo "my name is $name"";
};

##手册中的例子
一个基本的购物车，包括一些已经添加的商品和每种商品的数量。
其中有一个方法用来计算购物车中所有商品的总价格，该方法使
用了一个 closure 作为回调函数。

class Cart
{
    const PRICE_BUTTER  = 1.00;
    const PRICE_MILK    = 3.00;
    const PRICE_EGGS    = 6.95;
    protected $products = array();

    public function add($product, $quantity)
    {
        $this->products[$product] = $quantity;
    }

    public function getQuantity($product)
    {
        return isset($this->products[$product]) ? $this->products[$product] :FALSE;
    }

       public function getTotal($tax)
       {
        $total = 0.00;

        $callback =
        function ($quantity, $product) use ($tax, &$total)
        {
            $pricePerItem = 
            constant(__CLASS__ . 
            "::PRICE_" . strtoupper($product));
            $total += 
            ($pricePerItem * $quantity) * ($tax + 1.0);
        };
        array_walk($this->products, $callback);
        return round($total, 2);;
        }
    }
$my_cart = new Cart;
// 往购物车里添加条目
$my_cart->add('butter', 1);
$my_cart->add('milk', 3);
$my_cart->add('eggs', 6);
// 打出出总价格，其中有 5% 的销售税.
print $my_cart->getTotal(0.05) . "\n";
// 最后结果是 54.29
