Shell笔记

命令执行的顺序

1.执行使用绝对路径或相对路径执行的命令
2.使用alias
3.bash内部命令
4.安装环境变量$Path的命令

shell基础语法

定义变量

variableName=”value”
变量和等号之间不能有空格

经过定义的变量，只要在变量名前加上$就可使用

your_name="sun"
echo $your_name
echo ${your_name}

大括号可要可不要，但是下面这种情况需要

for skill in Ada Coffe Action Java
do
echo "I am good at ${skill}Script"
done

使用 readonly 命令可以将变量定义为只读变量，只读变量的值不能被改变。

readonly variableName

用unset可以删除变量

unset variableName

变量类型

  • 局部变量
  • 环境变量
  • shell变量

$表示当前shell的进程的ID，即pid

echo $$
